This directory contains GlassFish related packages,
which aim to be built separately from the main Jersey
project.

  - v2-package: contains a maven based project 
                to build Jersey package for GlassFish v2.x

  - v3-packages: contains a maven based project to build
                 OSGi modules and IPS packages for GlassFish v3
                 based GlassFish distribuitions 
                 These need to be built on JDK 1.6
