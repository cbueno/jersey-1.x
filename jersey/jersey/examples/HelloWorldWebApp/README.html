HelloWorldWebApp Example

This example demonstrates how to develop RESTful hello world web service with a Servlet 2.5 container.

Contents

The example consists of just one Java class:

com.sun.jersey.samples.helloworld.resources.HelloWorldResource
A resource class that produces a textual response to an HTTP GET.

The mapping of the URI path space is presented in the following table:

URI path Resource class HTTP methods
/helloworld HelloWorldResource GET

Running the Example

Run the example as follows:

Open the project using NetBeans 6.0 and deploy the project, you must have a servlet container registered in Netbeans to be able to deploy the application.

From a web browser, visit:

http://<host:port>/HelloWorldWebApp/helloworld
(for GlassFish V2: http://localhost:8080/HelloWorldWebApp/helloworld)
