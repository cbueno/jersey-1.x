Jersey is an implementation of JSR-311 JAX-RS: Java API for RESTful Web 
Services, a RESTful API that empowers/enables developers to rapidly build Web 
applications in Java that are characteristic of the best designed parts of the 
Web.

Type 'ant' to build the project.

Type 'ant dist' to build a distribution that is placed in the 'dist' directory.
The distribution included samples (which need to be compiled)
