 Enterprise Programming Model      
    
 The Java BluePrints projects presents the programming model, guidelines and examples for designing enterprise quality applications and web services using Java technologies. Some of the areas covered are Ajax-enabled Web 2.0 applications, Persistence,  JavaServer Faces, SOA with BPEL, and WS-Security.
